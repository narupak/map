ขั้นตอนการสร้างเว็บแผนที่ท่องเที่ยวภาคตะวันออก
	เราจะใช้server ที่เรียกว่า xampp เป็นการใช้คอมพิวเตอร์ของตนเองเป็น server 
	แผนที่ 
		1. จะต้องทำการเปิดใช้งาน Maps JavaScript Apiก่อน โดยเข้าไปเปิดใช้งานที่เว็บไซต์ "https://developers.google.com/maps/documentation/javascript/get-api-key"
		2. การเปิดใช้งานแผนที่ของ google เราจะต้องไป get key ที่จะถูกสร้างขึ้นจากเว็บ developers.google
		3. โดยการจะได้รับคีย์มาเราจะต้องเลือกว่าจะเปิดใช้งานอะไรบ้าง มี อยู่3 อย่าง
			1. Maps(แผนที่)
			2.Routes(เส้นทาง)
			3.Places(สถานที่)
		4.เมื่อเราเปิดใช้งานแล้วถ้าเราใช้งานเกิน 1 ปีจะต้องเสีย 900เหรียญ
		5.เมื่อเราได้คีย์มาแล้วจะต้องเอามาใส่ในส่วนของ YOUR_API_KEY ใน
		<script async defer src="https://maps.googleapis.com/maps/api/js?key=YOUR_API_KEY&callback=initMap"
		type="text/javascript"></script>
		ในส่วนนี้จะเป็นการเรียกใช้งานgoogleApi
	ฐานข้อมูล
		1.จะต้องสร้างฐานข้อมูลเอาไว้เก็บข้อมูลที่อยู่ในเว็บไซต์ ชื่อ maptravel
		2.สร้างตารางมา 3 ตารางคือ
			2.1 detail_map
				เป็นส่วนของการเก็บรายละเอียดของสถานที่ต่างๆ 
			2.2 location_map
				เป็นส่วนของการเก็บชื่อสถานที่และพิกัดของสถานที่
			2.3 travel_place
				เก็บในส่วนของรูปภาพและส่วนที่อ้างถึงของแต่ละสถานที่
	ทรัพยากรต่างที่จำเป็นของเว็บไซต์
		1. รูปภาพ
			นำรูปภาพมาไว้ในตำแหน่งที่เว็บสามารถเรียกดูได้ 
		2. style 
			เขียนstyle เพื่อที่จะตกแต่งหน้าเว็บ
	
	coding 
		เนื่องจากเว็บนี้เป็นการเขียนโดยใช้ framework ที่ชื่อว่า codeIgniter ซึ่ง codeIgniter จะมีไฟล์หลักที่ใช้อยู่ 3 ไฟล์ view,model,controller
		เริ่มต้นเราจะต้องสร้าง controller เพื่อที่จะแสดงหน้าเว็บเบราเซอร์ลงในเว็บไซต์
		โดยเราจะเรียกผ่าน $this->load->view("ที่อยู่ของไฟล์ที่เป็นหน้าเว็บ");
		เมื่อเราทำการสร้าง controller เสร็จเรียบร้อยเราจะต้องทดสอบเรียกว่าสามารถใช้งานได้หรือไม่ โดยจะเรียกผ่าน localhost/index.php/ตามด้วยชื่อของไฟล์ controller/ตามด้วยชื่อฟังชั่นที่สร้างใน controller 
		ถ้าสามารถเรียกหน้าวิวได้แล้วเราจะมาเริ่มต้นจากการเขียน html ก่อน
		HTML 
			จะเป็นส่วนของโครงสร้างเว็บไซต์ ส่วนของ tab ต่างๆ ให้สร้างtab รอไว้ตามจำนวนที่เราต้องการ ในที่นี้จะสร้างมา 5 แท็บ โดยจะประกอบไปด้วย หน้าแรก แหล่งข้อมูลท่องเที่ยว สถานที่ตั้ง แผนที่และเส้นทาง เกี่ยวกับโครงการวิจัย
				tab หน้าแรก
					จะเป็นส่วนของโชว์รูปภาพโดยจะสามารถเลื่อนรูปภาพได้ เมื่อคลิ๊กเข้าไปในภาพจะลิ้งไปเว็บไซต์ที่อ้างถึง
					โดยเราจะดึงข้อมูลจากฐานข้อมูลมาแสดงในslide ซึ่งประกอบไปด้วย ชื่อ รูปภาพ แหล่งอ้างอิง 
					เมื่อคลิ๊กจะทำการลิ้งไปตามแหล่งอ้างอิง
				tab แหล่งข้อมูลท่องเที่ยว
					จะเป็นส่วนที่จะแสดงข้อมูลของแต่ละสถานที่ ที่เราเลือกโดยการแสดงจะเป็นแบบ iframe คือการนำหน้าเว็บอื่นมาแสดงในขอบเขตที่เรากำหนด โดยเราใช้javascript ในการกำหนดตำแหน่งที่จะให้แสดงข้อมูล
				tab สถานที่ตั้ง
					จะแสดงรายละเอียดของสถานที่ตั้งต่างๆ ซึ่งมีทั้งหมด15สถานที่
				tab แผนที่และเส้นทาง
					เป็นส่วนของการแสดงแผนที่ ซึ่งเราจะเอาข้อมูลจากในฐานข้อมูลโดยประกอบด้วยสถานที่ตั้ง มาปักหมุดเพื่่อแสดงตำแหน่งสถานที่ต่างๆ 
					โดยเมื่อคลิ๊กที่ปักหมุดตั้งๆ จะแสดงรายละเอียดของสถานที่นั้นๆ และสามารถกดหาเส้นทางได้ โดยมันจะอ้างอิงจากตำแหน่งของเรากับสถานที่นั้นๆ
				tab เกี่ยวกับโครงการวิจัย
					เป็นส่วนของรายละเอียดของโครงการวิจับ
